What is this platform ?


This platform provides a tool for performing Sentiment Analysis. This platform allows the individual to answer typical interview questions by text. We have developped complex Deep Learning algorithm that analyze the results of the video or the text and provide the individual a summary of what our algorithm percieved, and how the individual performed compared to others.




Why this project ?


The project was developped as a college project for the B.Tech Final Project Evaluation. I have worked for several months on the individual models, and built this platform the final month. I am using Python for the individual model training, Flask for the web interface, and D3.js for visualization purposes.
